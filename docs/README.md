# Figmaデザイン関連ドキュメント

このディレクトリには、Figmaでトップページデザインを作成するためのすべての資料が含まれています。

## 📁 ファイル構成

### 1. `figma-design-spec.md`
**デザインシステムの詳細仕様書**

このファイルには以下の情報が含まれています：
- カラーパレット（プライマリ、セカンダリ、ステータス、グラデーション）
- タイポグラフィ（フォントサイズ、ウェイト、行間）
- スペーシング（基準値、スケール、コンポーネント内パディング）
- ボーダーとシャドウ
- レイアウト構造（ブレークポイント、コンテナ幅、グリッド）
- インタラクションとアニメーション
- アクセシビリティガイドライン
- Figma実装のヒント

### 2. `figma-design-tokens.json`
**Figma用デザイントークン（JSON形式）**

このファイルは以下のツールでインポート可能です：
- Figma Tokens Plugin
- Style Dictionary
- その他のデザイントークンツール

含まれるトークン：
- カラー（プライマリ、ニュートラル、セマンティック、グラデーション）
- タイポグラフィ（フォントファミリー、サイズ、ウェイト、行間、テキストスタイル）
- スペーシング（スケール、コンポーネント用）
- ボーダー半径
- シャドウ
- サイズ（アイコン、コンテナ）
- ブレークポイント
- アニメーション（継続時間、イージング、スケール）

### 3. `figma-component-spec.md`
**コンポーネントの詳細仕様書**

各コンポーネントの構造と仕様：
- ナビゲーションバー
- 変換履歴ボタン
- ヒーローアイコン
- ファイルアップロードエリア
- 変換ボタン
- メインカード
- フッター
- アラートメッセージ

各コンポーネントについて：
- 基本構造（ツリー表示）
- 詳細仕様（サイズ、色、フォントなど）
- バリアント（Default、Hover、Activeなど）
- Figma実装のベストプラクティス

## 🚀 使い方

### Step 1: デザイン仕様書を確認
`figma-design-spec.md`を読んで、デザインシステム全体を理解します。

### Step 2: デザイントークンをインポート
1. FigmaでFigma Tokensプラグインをインストール
2. プラグインを起動
3. `figma-design-tokens.json`をインポート
4. トークンからスタイル（Color Styles、Text Stylesなど）を生成

### Step 3: コンポーネントを作成
`figma-component-spec.md`を参照しながら、各コンポーネントを順番に作成します。

### Step 4: ページレイアウトを作成
1. デスクトップ、タブレット、モバイルのフレームを作成
2. 作成したコンポーネントを配置
3. オートレイアウトと制約を設定

### Step 5: プロトタイピング
インタラクションとアニメーションを設定します。

## 📝 注意事項

- すべてのカラー値は16進数で定義されています
- スペーシングの基準単位は4pxです
- タイポグラフィはシステムフォントを使用します
- グラデーションはFigmaのLinear Gradientで再現してください
- シャドウは複数レイヤーで表現することがあります
